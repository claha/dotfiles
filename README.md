# dotfiles

To install these dotfiles simply run:

```bash
stow emacs git i3 mpv qutebrowser
```

For everything to work correctly a few packages are needed. Depending on distro
these can be installed using apt, pacman etc.

- emacs
- feh
- git
- i3
- mpv
- qutebrowser
- ripgrep
- rxvt-unicode
- stow
- the silver searcher
- unclutter
- youtube-dl
